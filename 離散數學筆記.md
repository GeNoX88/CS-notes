第三章 向量空間

        |f1(x)          ...           fn(x)|
        |f1'(x)         ...          fn'(x)|
W(x) =  |                .                 |
        |                .                 |
        |                .                 |
        |f1^(n-1)(x)    ...     fn^(n-1)(x)|

一組具相同定義域的函數集合{f1,...,fn}，若其Wronskian於某個x值為零，即W(x) ≠ 0 則這組函數為線性獨立 ☆W(x) = 0 卻不一定是線性相依

t_0~t_n為F中n+1個相異元素，則
f_i(x) = ∏((x-t_j)/(t_i-t_j))|j≠i, i=0, 1, ..., n
稱為相對於t_i的Lagrange polynomials
